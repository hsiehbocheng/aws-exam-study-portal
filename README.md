# AWS AIP-C01 考題練習系統

本機 Web 應用程式，用於 AWS AI Practitioner (AIP-C01) 認證考試的學習與模擬測驗。

## 功能

| 模式 | 說明 |
|------|------|
| **Study** | 依序瀏覽題目、選擇答案後顯示解析、中英對照翻譯、打星收藏、筆記、一鍵複製題目 |
| **Exam** | 模擬真實考試：題目與選項隨機排列、180 分鐘倒數計時、交卷後顯示成績與錯題詳情 |
| **List** | 題目總覽清單，可篩選打星題目、展開查看選項與解析、支援中文翻譯 |

### 額外功能

- **答案修正**：可針對單題修正正確答案，修正後的答案會用於後續考試評分
- **答案差異標示**：自動標記社群答案 (Community) 與最終答案 (Final) 不同的題目
- **考試歷史**：保留每次考試紀錄，可回顧每題的作答與正確答案

## 環境需求

- Python 3.12+
- [uv](https://docs.astral.sh/uv/)（Python 套件管理工具）

## 快速開始

### 1. 準備題庫檔案

將題庫 `.xlsx` 檔案放在本目錄下，命名為：

```
AIP-C01-questions-explain.xlsx
```

檔案格式為 Excel (.xlsx)，第一列為標題列，第二列起為題目資料。需包含以下欄位：

| 欄位（從 A 開始） | 內容 | 必填 |
|---|---|---|
| A (第 1 欄) | 題號 | ✅ |
| B (第 2 欄) | 英文題目 | ✅ |
| C (第 3 欄) | 英文選項（格式：`A. 文字`、`B. 文字`⋯ 以換行分隔） | ✅ |
| D (第 4 欄) | 社群答案（如 `A`、`BD`、`ADF`） | ✅ |
| E (第 5 欄) | UID | |
| G (第 7 欄) | Topic 編號 | |
| H (第 8 欄) | 參考連結 | |
| I (第 9 欄) | 中文題目 | |
| J (第 10 欄) | 中文選項（格式同英文選項） | |
| N (第 14 欄) | 解題邏輯 | |
| O (第 15 欄) | 各選項解析（格式：`A: 解析文字`、`B: 解析文字`⋯ 以換行分隔） | |
| P (第 16 欄) | 最終答案（優先使用；為空時回退到第 4 欄社群答案） | |

### 2. 安裝依賴與建立虛擬環境

```bash
cd aws-aip-quiz
uv sync
```

### 3. 匯入題庫

```bash
uv run python3 init_db.py
```

此步驟會讀取上層目錄的 `AIP-C01-questions-explain.xlsx`，將題目匯入 `quiz.db`。

> 重新執行 `init_db.py` 只會更新題目資料，**不會**清除你的打星、筆記、作答紀錄與答案修正。

### 4. 啟動服務

```bash
uv run flask run
```

開啟瀏覽器前往 http://127.0.0.1:5000 即可使用。

```
    └── templates/

```
